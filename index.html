design

APIs are lightweight and modular, making porting faster and game instances leaner
We target snapshots as well, allowing for snappier mod updates and more informed community
All parts of Fabric are fully free for everyone to use to any extent!

minecraft modding toolchain

projects

core

  • Fabric Loader

    Version-independent mod loader, providing mod loading and code patching functionality

  • Fabric API

    “Hook layer”, providing the most common hooks and interoperability required for Fabric mods

  • Fabric Installer

    Tool for creating Fabric-ready profiles for the official Minecraft launcher

  • Loom

    Gradle-based build system, used for developing Fabric mods

  • Yarn

    Deobfuscated friendly names for the most recent versions of the game based on community contributions

  • Intermediary

    Stable cross-version names and match information between Minecraft releases

libs

  • Fabric Language Kotlin

    Fabric Loader module for supporting Kotlin language in mods

  • Fabric Language Scala

    Fabric Loader module for supporting Scala language in mods

  • Fabric Language Groovy

    Fabric Loader module for supporting Groovy language in mods

  • Mixin

    Trait/mixin framework for Java created by Mumfrey for Sponge - forked by Fabric to adapt to our use cases

  • tiny-mappings-parser

    Library for processing Tiny-format mapping files

tools

  • Enigma

    Tool for deobfuscation of Java classes originally by Cuchaz Interactive . Our changes include countless bugfixes, optimizations and improvements across the codebase, as well as user experience and compatibility improvements.

    Downloads can be found here

  • Stitch

    Assortment of small tools providing various functionality used by the Fabric project in command-line form, such as generating and updating “intermediaries” for cross-version name stability, based on Matcher output or merging client and server Minecraft .JARs.

  • Weave

    Earlier iteration of Stitch and is now generally only used to export Enigma-format mappings to Tiny-format files

tools (continuation)

  • tiny-remapper

    tiny-remapper project provides a fast, optimized tool for accurately remapping JAR files using the Tiny mapping format.

  • Matcher

    sfPlayer1’s tool for tracking elements in obfuscated Java archives across releases used for updating Yarn mappings

  • Fernflower

    Modified version of JetBrains’ upstream with line number mapping information exposed

  • Procyon

    Modified version of Procyon decompiler with minor bug fixes

  • CFR

    Modified version of CFR Java decompiler with minor code generation quality extensions

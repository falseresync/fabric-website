download

Fabric consists of 2 parts: Fabric Loader and Fabric API. You need both for most mods to run. Choose the launcher you would like to use below and follow the instructions nearby.
Download Fabric Installer (Windows)
Download Fabric Installer (Any OS)
Download Fabric API

install

The Minecraft version can be selected in the installer, this download works for every version we support.
1
Open the Installer. In the window you need to configure the game and loader versions and the install location.
  • Generally using the latest available Loader version is recommended.
  • The default installation location should be fine on most platforms.
2
Press Install. A new game version and profile will be created in the launcher's menu, which you can now use to launch Fabric.
3
Put Fabric API into .minecraft/mods/ folder.

join our community

goals and ideas

APIs are lightweight and modular, making porting faster and game instances leaner
We target new snapshots as well as releases, allowing for snappier mod updates and more informed community
All Fabric projects are free and open-source, and anyone can contribute!
We strive to be unopinionated to our players and modders. There is no "the way" we're forcing you to follow.

make mods

Making mods with Fabric is easy and fun! We have a whole community of modders at our Discordready to help you if you struggle. But you need to start somewhere. Here's a couple of steps to get you going.
1
Check out this guide on our wikion how to set up your development environment.
2
Try adding an itemor a block.Both of the tutorials you can find on our wiki by the provided links.
3
Explore the wiki! You will find all sorts of stuff there. It might be a little out of date, so if something just doesn't wanna work, don't hesitate to ask questions. You're always welcome!

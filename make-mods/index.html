new to modding?

Making mods with Fabric is easy and fun! We have a whole community of modders at our Discordready to help you if you struggle. But you need to start somewhere. Here's a couple of steps to get you going.
1
Check out this guide on our wikion how to set up your development environment.
2
Try adding an itemor a block.Both of the tutorials you can find on our wiki by the provided links.
3
Explore the wiki! You will find all sorts of stuff there. It might be a little out of date, so if something just doesn't wanna work, don't hesitate to ask questions. You're always welcome!

want examples?

1
Check out Fabric Example Mod providing you a configured bare minimum template to start modding.
2
There's Fabric Example Mod in Kotlintoo!
3
If you are into big mods, following mods and libs are a great place for studying:
  • TechReborn org
  • TerraformersMC org
  • REI mod
4
Struggling with Mixins? Look at SpongePowered projects code for examples! Though those aren't Fabric, they still contain a lot of applicable reference examples.

dev env versions

The recommended Loom version is 0.4-SNAPSHOT
Fabric API version may not be accurate. Try consulting the CurseForge pageif you have troubles. If you still cannot resolve a correct version of it, ask for help in our Discord.
minecraft_version=[minecraftVersion] yarn_mappings=[yarnVersion] loader_version=[loaderVersion] fabric_version=[fabricVersion]

mappings migration

Mappings can be auto updated by using the following command. Consult the wiki pagefor more information.
gradlew migrateMappings --mappings [minecraftVersion]
